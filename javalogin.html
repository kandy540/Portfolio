Java LogIn-GUI

Developed a Java system where an user can type in their username and password to get notified that they have successfully loged in. However, if the user types in the wrong username or password they will an error saying that they typed in the wrong username or password. Also, if the user leaves out any text field blank, they will get an error saying that they will need to fill it in.
